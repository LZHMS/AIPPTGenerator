# AI PPT Generator - 智能PPT生成器

基于 LangGraph 的 AI 驱动 PPT 自动生成工具。

## 功能特点

- 🎯 **智能主题生成**: 根据输入主题自动生成合适的 PPT 风格
- 🎨 **多种配色方案**: 商务蓝、科技紫、自然绿等6种主题
- 🤖 **AI 内容生成**: 使用 LLM 直接生成高质量内容
- 📊 **15种布局类型**: 时间线、对比、统计等多样化布局
- 📥 **一键下载**: 生成的 PPT 可直接下载

## 使用方法

1. 输入 PPT 主题
2. 选择页数（4-20页）和风格
3. 点击生成
4. 下载 PPT 文件

## 环境变量

需要配置 `MOONSHOT_API_KEY`（在 Space Settings → Secrets 中添加）

获取 API Key: https://platform.moonshot.cn/
